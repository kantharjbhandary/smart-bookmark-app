# Smart Bookmark App

## Features
- Google OAuth Login
- Private Bookmarks with Supabase RLS
- Realtime updates
- Next.js App Router

## Setup
1. Add .env.local
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=

2. npm install
3. npm run dev
